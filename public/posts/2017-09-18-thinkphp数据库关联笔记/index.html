thinkPHP数据库关联笔记

class News extends Base{
public function user() { return $this-&gt;belongsTo(‘MemberList’,’news_auto’, ‘member_list_id’); } }</code></pre></p><p><span>控制器中调用用例:</span></p><p><pre lay-lang="JavaScript"><code class="JavaScript">namespace app\home\controller;
