Deprecated: Directive 'track_errors' is deprecated in Unknown on line 0

问题

在命令行运行php命令的时候，报如下提示： Deprecated: Directive ’track_errors’ is deprecated in Unknown on line 0

解决方法

打开php.ini，搜索track_errors，

track_errors=On

改成

track_errors=Off

然后重启nginx或者apache,重新运行：
