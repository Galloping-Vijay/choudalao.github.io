PHP 汉字转拼音工具

<p>基于&nbsp;<a href="http://cc-cedict.org/wiki/" _href="http://cc-cedict.org/wiki/">CC-CEDICT</a>&nbsp;词典的中文转拼音工具，更准确的支持多音字的汉字转拼音解决方案。</p><h2><a id="user-content-安装" class="anchor" href="https://github.com/overtrue/pinyin#%E5%AE%89%E8%A3%85" _href="https://github.com/overtrue/pinyin#%E5%AE%89%E8%A3%85"></a>安装</h2><p><br></p><p>使用 Composer 安装:</p><p><br></p><pre lay-lang="JavaScript"><code class="JavaScript">composer require "overtrue/pinyin:~3.0"</code></pre><h2><br></h2><h2>使用</h2><p><br></p><h2><a id="user-content-使用" class="anchor" href="https://github.com/overtrue/pinyin#%E4%BD%BF%E7%94%A8" _href="https://github.com/overtrue/pinyin#%E4%BD%BF%E7%94%A8"></a></h2><h3><a id="user-content-拼音数组" class="anchor" href="https://github.com/overtrue/pinyin#%E6%8B%BC%E9%9F%B3%E6%95%B0%E7%BB%84" _href="https://github.com/overtrue/pinyin#%E6%8B%BC%E9%9F%B3%E6%95%B0%E7%BB%84"></a>拼音数组</h3><p><br></p><pre lay-lang="JavaScript"><code class="JavaScript">use Overtrue\Pinyin\Pinyin;

$pinyin = new Pinyin(); $pinyin-&gt;convert(‘带着希望去旅行，比到达终点更美好’); // ["dai", "zhe", "xi", "wang", "qu", "lu", "xing", "bi", "dao", "da", "zhong", "dian", "geng", "mei", "hao"] $pinyin-&gt;convert(‘带着希望去旅行，比到达终点更美好’, PINYIN_UNICODE); // ["dài","zhe","xī","wàng","qù","lǚ","xíng","bǐ","dào","dá","zhōng","diǎn","gèng","měi","hǎo"] $pinyin-&gt;convert(‘带着希望去旅行，比到达终点更美好’, PINYIN_ASCII); //["dai4","zhe","xi1","wang4","qu4","lv3","xing2","bi3","dao4","da2","zhong1","dian3","geng4","mei3","hao3"]</code></pre><p>选项：</p><table><thead><tr class="firstRow"><th>选项</th><th>描述</th></tr></thead><tbody><tr><td><code>PINYIN_NONE</code></td><td>不带音调输出:&nbsp;<code>mei hao</code></td></tr><tr><td><code>PINYIN_ASCII</code></td><td>带数字式音调：&nbsp;<code>mei3 hao3</code></td></tr><tr><td><code>PINYIN_UNICODE</code></td><td>&nbsp;式音调：<code>měi hǎo</code></td></tr></tbody></table><h3><a id="user-content-生成用于链接的拼音字符串" class="anchor" href="https://github.com/overtrue/pinyin#%E7%94%9F%E6%88%90%E7%94%A8%E4%BA%8E%E9%93%BE%E6%8E%A5%E7%9A%84%E6%8B%BC%E9%9F%B3%E5%AD%97%E7%AC%A6%E4%B8%B2" _href="https://github.com/overtrue/pinyin#%E7%94%9F%E6%88%90%E7%94%A8%E4%BA%8E%E9%93%BE%E6%8E%A5%E7%9A%84%E6%8B%BC%E9%9F%B3%E5%AD%97%E7%AC%A6%E4%B8%B2"></a>生成用于链接的拼音字符串</h3><p><br></p><pre lay-lang="JavaScript"><code class="JavaScript">$pinyin-&gt;permlink(‘带着希望去旅行’); // dai-zhe-xi-wang-qu-lu-xing </code></pre><pre lay-lang="JavaScript"><code class="JavaScript">$pinyin-&gt;permlink(‘带着希望去旅行’, ‘.’); // dai.zhe.xi.wang.qu.lu.xing</code></pre><h3><br></h3><h3><a id="user-content-获取首字符字符串" class="anchor" href="https://github.com/overtrue/pinyin#%E8%8E%B7%E5%8F%96%E9%A6%96%E5%AD%97%E7%AC%A6%E5%AD%97%E7%AC%A6%E4%B8%B2" _href="https://github.com/overtrue/pinyin#%E8%8E%B7%E5%8F%96%E9%A6%96%E5%AD%97%E7%AC%A6%E5%AD%97%E7%AC%A6%E4%B8%B2"></a>获取首字符字符串</h3><p><br></p><pre lay-lang="JavaScript"><code class="JavaScript">$pinyin-&gt;abbr(‘带着希望去旅行’); // dzxwqlx $pinyin-&gt;abbr(‘带着希望去旅行’, ‘-’); // d-z-x-w-q-l-x</code></pre><p><br></p><h3><a id="user-content-翻译整段文字为拼音" class="anchor" href="https://github.com/overtrue/pinyin#%E7%BF%BB%E8%AF%91%E6%95%B4%E6%AE%B5%E6%96%87%E5%AD%97%E4%B8%BA%E6%8B%BC%E9%9F%B3" _href="https://github.com/overtrue/pinyin#%E7%BF%BB%E8%AF%91%E6%95%B4%E6%AE%B5%E6%96%87%E5%AD%97%E4%B8%BA%E6%8B%BC%E9%9F%B3"></a>翻译整段文字为拼音</h3><p><br></p><p>将会保留中文字符：<code>，。 ！ ？ ： “ ” ‘ ’</code>&nbsp;并替换为对应的英文符号。</p><p><br></p><pre lay-lang="JavaScript"><code class="JavaScript">$pinyin-&gt;sentence(‘带着希望去旅行，比到达终点更美好！’); // dai zhe xi wang qu lv xing, bi dao da zhong dian geng mei hao! $pinyin-&gt;sentence(‘带着希望去旅行，比到达终点更美好！’, true); // dài zhe xī wàng qù lǚ xíng, bǐ dào dá zhōng diǎn gèng měi hǎo!</code></pre><p><br></p><h3><a id="user-content-翻译姓名" class="anchor" href="https://github.com/overtrue/pinyin#%E7%BF%BB%E8%AF%91%E5%A7%93%E5%90%8D" _href="https://github.com/overtrue/pinyin#%E7%BF%BB%E8%AF%91%E5%A7%93%E5%90%8D"></a>翻译姓名</h3><p><br></p><p>姓名的姓的读音有些与普通字不一样，比如 ‘单’ 常见的音为&nbsp;<code>dan</code>，而作为姓的时候读&nbsp;<code>shan</code>。</p><p><br></p><pre lay-lang="JavaScript"><code class="JavaScript">$pinyin-&gt;name(‘单某某’); // [‘shan’, ‘mou’, ‘mou’] $pinyin-&gt;name(‘单某某’, PINYIN_UNICODE); // ["shàn","mǒu","mǒu"]</code></pre><p><br></p><p><br></p>
