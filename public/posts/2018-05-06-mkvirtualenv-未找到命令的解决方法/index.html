mkvirtualenv： 未找到命令的解决方法

<p style="text-align: justify;">1.升级python包管理工具pip<span>(如果您的pip可以用,不建议升级,因为升级后可能和您的python版本的包不匹配,导致无法使用,所以,pip可以用的话,跳过步骤1)</span></p><p style="text-align: justify;"><pre lay-lang="JavaScript"><code class="JavaScript">pip install –upgrade pip</code></pre></p><p style="text-align: justify;"><span>备注：当你想升级一个包的时候 pip install --upgrade 包名</span></p><p style="text-align: justify;"><span><br></span></p><p style="text-align: justify;">2.python虚拟环境安装</p><pre lay-lang="JavaScript"><code class="JavaScript">sudo apt-get install python-virtualenv sudo easy_install virtualenvwrapper</code></pre><p style="text-align: justify;">上述工具装好后找不到mkvirtualenv命令，需要执行以下环境变量设置。</p><pre lay-lang="JavaScript"><code class="JavaScript">1.创建目录用来存放虚拟环境 mkdir $HOME/.virtualenvs 2.在~/.bashrc中添加行： export WORKON_HOME=$HOME/.virtualenvs source /usr/local/bin/virtualenvwrapper.sh 3.运行: source ~/.bashrc</code></pre><p style="text-align: justify;">3.创建python虚拟环境</p><pre lay-lang="JavaScript"><code class="JavaScript">mkvirtualenv [虚拟环境名称] workon [虚拟环境名称]</code></pre><p style="text-align: justify;">4.退出虚拟环境 离开 deactivate</p><p style="text-align: justify;">5.删除(慎用)&nbsp;<br>rmvirtualenv [虚拟环境名称]</p><p><br></p>
