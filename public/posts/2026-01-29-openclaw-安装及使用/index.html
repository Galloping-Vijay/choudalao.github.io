openclaw 安装及使用

项目概述

openclaw 是一个开源的 AI 助手框架，提供你自己的个人 AI 助手。

  • 标语: Your own personal AI assistant. Any OS. Any Platform. The lobster way. 🦞
  • 特点:
    • 支持任何操作系统
    • 支持任何平台
    • 采用"龙虾方式"（The lobster way）

官方链接

  • GitHub 仓库: https://github.com/moltbot/moltbot

📝 重要说明

由于 clawdbot 被 Claude 母公司起诉，现已改名为 moltbot，但命令行界面存在一定混乱。实际上安装的是 clawdbot 而不是 moltbot，官网文档也存在这个问题。

这是第三次改名了,现在名称是openclaw

可以通过查看 npm 包环境变量来确认安装的版本：

注意: 请根据自己实际安装的版本来执行相应的命令。

安装步骤

步骤 1：全局安装 openclaw

Windows 环境安装（推荐）

方法一：官方安装脚本（推荐）
# 以管理员身份运行 PowerShell
iwr -useb https://molt.bot/install.ps1 | iex
方法二：npm 全局安装
npm install -g openclaw@latest

验证安装:

clawdbot --version

初始化和配置守护进程

clawdbot onboard --install-daemon

手动运行

clawdbot gateway --port 18789 --verbose

Gateway 服务管理

常用命令：

# 开启守护进程
clawdbot gateway start
# 查看状态
clawdbot gateway status

# 停止守护进程
clawdbot gateway stop

# 重启
clawdbot gateway restart

# 查看帮助
clawdbot gateway --help

📱 第三方平台配置

WhatsApp 登录

除了在初始化的时候配置，还可以执行下面命令

clawdbot channels login

Telegram 配置

步骤 1：在 Telegram 中创建机器人

  1. 打开 Telegram
  2. 与 @BotFather 对话
  3. 点击 Create a New Bot
  4. 输入机器人名称
  5. 复制生成的 token

步骤 2：配置到 clawdbot

clawdbot channels add --channel telegram --token "您的Token"

故障排除：配置文件编辑

如果出现错误，可以找到 clawdbot.json 文件，在 channels 部分添加以下配置：

{
  "telegram": {
    "enabled": true,
    "dmPolicy": "pairing",
    "botToken": "xxx",
    "groupPolicy": "allowlist",
    "streamMode": "partial"
  }
}

完成后: 搜索机器人名称并发送消息即可开始使用。
