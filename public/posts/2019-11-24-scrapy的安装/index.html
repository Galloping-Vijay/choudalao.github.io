Scrapy的安装

Scrapy的安装介绍

windows

查看win下python版本

升级pip版本：pip install --upgrade pip

通过pip 安装 Scrapy 框架 pip install Scrapy

ubuntu

如果没有安装pip3先安装,sudo apt install python3-pip. 安装非Python的依赖 sudo apt-get install python-dev python-pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev

sudo pip3 install scrapy

查看版本 scrapy version
