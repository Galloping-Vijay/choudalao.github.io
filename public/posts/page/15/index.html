随着我们提交代码次数的增多，你会发现 Git 包变得越来越臃肿，尤其是上传大文件上去的时候，当你删除文件时，其实 .git目录下的大文件对象是不会被删除的。 Git 会把文件的每一个差异化版本都记录在案。这意味着，即使你只改动了某个文件的一行内容，Git 也会生成一个全新的对象来存储新的文件内容。

Python 获取某天是星期几

代码

from datetime import datetime
import time

gap = 1
# 时间戳
day = time.time() - 86400*(gap-1)
# 输出年月日
ymd = time.strftime('%Y-%m-%d', time.localtime(day))
# 获取星期几 数字1-7代表周一到周日
week = datetime.strptime(ymd, '%Y-%m-%d').isoweekday()
