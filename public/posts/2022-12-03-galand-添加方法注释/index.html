如何安装插件

file --> settings --> plugins

搜索 Goanno,安装

设置模板

Tools --> Goanno Setting

模板示例

// ${function_name} 
// @Summary ${todo}
// @Receiver ${receiver}
// @Param ${params}
// @Return ${return_types}
// @Router /api/v1/xxx [get]
// @Author Vijay

使用

1.快捷键ctrl + commond + / （ windows系统 ctrl + alt + /）
2.手动生成：右键 -> Generate -> Goanno

修改或查看快捷键
