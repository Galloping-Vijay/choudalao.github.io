InvalidArgumentException Please provide a valid cache path

InvalidArgumentException：Please provide a valid cache path

这是因为laravel的缓存路径没有找到

laravel缓存文件路径是在 config/cache.php中设置，默认存在storage文件夹中

解决：

需要保证storage/framework下面创建 sessions， views, cache 文件夹并确保可写权限
