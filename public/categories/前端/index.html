在 Windows 下可以安装多个 Node.js 版本

使用 nvm-windows 管理多个 Node.js 版本

安装 nvm-windows:

下载并安装 nvm-windows 安装完成后，打开命令行工具（如 PowerShell 或 CMD）。

列出可下载的版本号

nvm list available

安装多个 Node.js 版本

使用以下命令安装不同版本的 Node.js：

nvm install 14.16.0  # 安装 Node.js 14.16.0
# nvm install 14.16.0 --reinstall-packages-from=14.16.0
nvm install 18.17.1  # 安装 Node.js 18.17.1

查看列表

nvm list

切换 Node.js 版本：

使用以下命令切换到指定版本：

GET http://xxx:8080/sockjs-node/info?t=1650300806666
