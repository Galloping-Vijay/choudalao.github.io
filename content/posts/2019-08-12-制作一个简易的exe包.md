# 需求
我们有个pc网页应用,用户希望有个win包,点击点开的同时,浏览器全屏展示

# 实现
### 制作bat文件
创建一个 臭大佬.bat 文件
```go
@echo off
:: 打开链接
start "" "https://www.choudalao.com/"
:: 模拟f11全屏打开
powershell -Command "& {(New-Object -ComObject WScript.Shell).SendKeys('{F11}');}"
```

### bat转换为exe
Bat To Exe Converter 是一款可以将批处理文件bat转换成exe应用程序的软件。
软件下载：http://www.pc6.com/softview/SoftView_31106.html

安装后,打开软件,界面中打开我们的bat文件,修改版本图标的信息,然后保存转换.
