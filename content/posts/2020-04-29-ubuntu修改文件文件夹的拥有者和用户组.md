# 说明
我们以.git目录的文件为例，我们把里面用户组和用户为root的，都改成www。我们先看一下目录的文件情况。
```shell
ll
```
![](https://www.choudalao.com/uploads/20200429/20200429225458GEX1V0.png)
可以看到，有两个文件是root的，我们来修改它。


# 切换用户
```shell
su root
```
# 修改文件（目录）所有者
```shell
chown www index
//如果是目录
//chown -R www index/
```
R：递归修改目录下所有的拥有者
www:拥有者名字
index/:更改目录

### 修改文件（目录）用户组
```shell
chgrp www index
// 如果index是目录
//chgrp -R www index/
```
