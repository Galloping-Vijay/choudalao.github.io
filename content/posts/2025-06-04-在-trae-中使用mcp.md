# 前言
之前我们探索过 [Cline中使用 MySql MCP 服务](https://www.choudalao.com/article/337 "Cline中使用 MySql MCP 服务"),最近使用 Trae 开发项目,里面继承了很多 MCP 服务,今天我们来试试在 Trae 中使用 MCP。

# 安装
打开 MCP 菜单，我们也以 mysql 为例，在搜索栏中搜索 mysql，点击设置，查看介绍页面。
![](https://www.choudalao.com/uploads/20250604/20250604165007WC63VW.png)
![](https://www.choudalao.com/uploads/20250604/20250604165035pnseKZ.png)

根据文档说明，我们本地有 python 环境，我们用pip手动安装
```go
pip install mysql-mcp-server
```
![](https://www.choudalao.com/uploads/20250604/20250604165200HuSnCW.png)

# 配置
找到`mysql-mcp-server` 安装目录,写入配置如下:
```go
{
  "mcpServers": {
    "test-db": {
      "command": "uv",
      "args": [
        "--directory",
        "E:\\xxx",
        "run",
        "mysql_mcp_server"
      ],
      "env": {
        "MYSQL_HOST": "xxx",
        "MYSQL_PORT": "3306",
        "MYSQL_USER": "xxx",
        "MYSQL_PASSWORD": "xxx",
        "MYSQL_DATABASE": "xxx"
      }
    }
  }
}
```
如果配置没有问题的话，会显示可用标识：
![](https://www.choudalao.com/uploads/20250604/20250604165537mTUHeU.png)
配置好后，我们来试一试
