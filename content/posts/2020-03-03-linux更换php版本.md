# 查看当前版本
```php
php -v
```
![](https://www.choudalao.com/uploads/20200303/20200303221733l4ydGK.png)

# 首先你本地有已经下好的新版本
博主本地已有php7.2的
![](https://www.choudalao.com/uploads/20200303/20200303221920TsBSxV.png)

# 删除掉默认的
```php
rm -f /usr/bin/php
```
# 引用你想要的版本

```php
ln -sf /www/server/php/72/bin/php /usr/bin/php
```
