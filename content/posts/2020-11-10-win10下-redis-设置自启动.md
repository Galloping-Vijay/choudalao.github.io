# 问题
win下安装redis后，需要 cmd 启动，只要一退出命令窗口，redis 就消失了。

![](https://www.choudalao.com/uploads/20201110/20201110111348H4YYYW.png)

所以，需要把redis服务添加到后台服务，切换到redis安装的根目录，命令如下：

```shell
// redis-server --service-install redis.windows-service.conf --loglevel verbose
.\redis-server.exe --service-install redis.windows-service.conf --loglevel verbose
```
![](https://www.choudalao.com/uploads/20201110/20201110111525RWjNRR.png)

![](https://www.choudalao.com/uploads/20201110/20201110111823FbmTey.png)

如果要结束的话，执行

```shell
redis-server.exe --service-stop
```

常用的redis服务命令。

卸载服务：redis-server --service-uninstall

开启服务：redis-server --service-start

停止服务：redis-server --service-stop
