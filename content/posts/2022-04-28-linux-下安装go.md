# 下载

安装包下载地址为：https://golang.org/dl/。

如果打不开可以使用这个地址：https://golang.google.cn/dl/。

![](https://www.choudalao.com/uploads/20220428/20220428195127uKo5vi.png)

复制这边的链接，下载

```go
wget https://dl.google.com/go/go1.18.1.linux-amd64.tar.gz
```

# 解压

```go
tar -C /usr/local/ -zxvf go1.18.1.linux-amd64.tar.gz
```

# 添加系统环境变量

##### 创建文件
```go
vim /etc/profile.d/go.sh
```

##### 写入
```go
export PATH=$PATH:/usr/local/go/bin
```

##### 配置文件生效

```go
source /etc/profile.d/go.sh
```

# 设置 GOPATH 目录

##### 创建工作目录
```go
mkdir -p /www/wwwroot/go
```
##### 创建gopath目录
```go
mkdir -p /www/wwwroot/go/goPath
```
##### 将目录添加到GOPATH中

```go
vim /etc/profile.d/gopath.sh
```

###### GOPATH具体指向位置

```go
export GOPATH=/www/wwwroot/go/goPath
```

###### 使配置文件生效
```go
source /etc/profile.d/gopath.sh
```

# 验证
```go
echo $GOPATH
```

```go
go env
```

```go
go version
```

// 开启GO111MODULE
```go
go env -w GO111MODULE=on
// 关闭
// go env -w GO111MODULE=off
//  自己检测是不是使用 modules 功能
// go env -w GO111MODULE=auto
```

// 设置代理
```go
go env -w GOPROXY=https://goproxy.cn,direct
```
