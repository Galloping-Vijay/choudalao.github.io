# 问题
go 在引入包的时候经常能看到以下几种写法：

```php
import (
	"github.com/astaxie/beego"
	f"fmt"
	_ "blogApi/routers"
	. "blogApi/models"
)
```

# 解释
**f"fmt"** ：表示别名，使用别名后
`fmt.Println("hello world")`可以写成`f.Println("hello world")`

**下划线**：使用下划线在只会调用包里的init函数，别的函数不会被导入。

**点开头**：这个包导入之后在你调用这个包的函数时，你可以省略前缀的包名。如 `import（.“fmt”）`时，原来`fmt.Println("hello world")`可以写成`Println("hello world")`
