# 安装
composer 安装
```shell
composer require "overtrue/laravel-lang:~3.0"
```

![](https://www.choudalao.com/uploads/20191130/20191130211606njavY2.png)

# 修改 config/app.php 配置文件
将Illuminate\Translation\TranslationServiceProvider::class, 替换为 Overtrue\LaravelLang\TranslationServiceProvider::class,，然后将 'locale' => 'en', 修改成 'locale' => 'zh-CN',
![](https://www.choudalao.com/uploads/20191130/20191130211802Z4lF1E.png)
![]![](https://www.choudalao.com/uploads/20191130/20191130212004cuNAgA.png)
# 执行
执行命令
```shell
php artisan lang:publish zh-CN
```
