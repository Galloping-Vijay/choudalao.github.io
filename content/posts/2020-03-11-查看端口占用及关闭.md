# linux
## 使用netstat命令：

查看已经连接的服务端口（ESTABLISHED）
```python
netstat -a
```
查看所有的服务端口（LISTEN，ESTABLISHED）
```python
netstat -anp
```
查看指定端口，可以结合grep命令：
```php
netstat -anp | grep 8080
```
也可以使用lsof命令：
```php
lsof -i:8888
```
若要关闭使用这个端口的程序，使用kill + 对应的pid
```json
kill -9 PID号
```
ps：kill就是给某个进程id发送了一个信号。默认发送的信号是SIGTERM，而kill -9发送的信号是SIGKILL，即exit。exit信号不会被系统阻塞，所以kill -9能顺利杀掉进程。

# win10
查看端口占用
```json
netstat -ano |findstr -C:端口
```
查看端口占用的pid
```json
tasklist | findstr -C:端口
```
使用taskkill强制终止指定PID的进程
```json
taskkill -F /PID  进程id
```
