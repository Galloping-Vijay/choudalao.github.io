# 说明

GitBook 是一个基于 Node.js 的命令行工具，支持 Markdown 和 AsciiDoc 两种语法格式，可以输出 HTML、PDF、eBook 等格式的电子书。

GitBook 不是 Markdown 编辑工具，也不是 Git 版本管理工具。

但 GitBook 又与 Markdown 和 Git 息息相关，因为只有将它们结合起来使用，才能将它们的威力发挥到极致！因此，通常我们会选择合适的 Markdown 编辑工具以获得飞一般的写作体验；使用 GitBook 管理文档，预览、制作电子书；同时通过 Git 管理书籍内容的变更，并将其托管到云端（比如 GitHub、GitLab、码云，或者是自己搭建的 Git 服务器），实现多人协作。

因此，我推荐，也是我目前使用的搭配是 GitBook + Typora + Git。

# 安装

详细步骤参考[官网](https://github.com/GitbookIO/gitbook "官网")，应该被墙了。

需要安装 nodejs，以便能够使用 npm 来安装 gitbook。下载node，[点击这里](https://nodejs.org/en/download/ "点击这里")

安装gitbook命令：

```shell
npm install -g gitbook-cli
```

安装成功可以打印版本查看：

```shell
gitbook --version
```

![](https://www.choudalao.com/uploads/20200110/20200110235705aXMBHT.png)

推荐安装 `Typora`作为gitbook的编辑器（一个很棒的支持 macOS、Windows、Linux 的 Markdown 编辑工具）

Typora 下载地址：https://typora.io/
Git 下载地址：https://git-scm.com/downloads

# 使用

使用 gitbook init 初始化书籍目录：

```shell
gitbook init
```
![](https://www.choudalao.com/uploads/20200111/202001110002155GmfSr.png)

![](https://www.choudalao.com/uploads/20200111/20200111000223zHrH42.png)

README.md —— 书籍的介绍写在这个文件里
SUMMARY.md —— 书籍的目录结构在这里配置

使用`Typora`编辑这两个文件

![](https://www.choudalao.com/uploads/20200111/20200111000400IdL1SX.png)

可以在`SUMMARY.md`里面加我们需要的目录大纲

![](https://www.choudalao.com/uploads/20200111/20200111001339pzjega.png)

再次执行`gitbook init`命令，GitBook 会查找 SUMMARY.md 文件中描述的目录和文件，如果没有则会将其创建。

![](https://www.choudalao.com/uploads/20200111/20200111001514t0fGM6.png)

![](https://www.choudalao.com/uploads/20200111/20200111001526NGtZ9u.png)


书籍目录结构创建完成以后，就可以使用 gitbook serve 来编译和预览书籍了

```shell
 gitbook serve
```

![](https://www.choudalao.com/uploads/20200111/20200111001753L3kBoB.png)

![](https://www.choudalao.com/uploads/20200111/20200111001805ES714d.png)

![](https://www.choudalao.com/uploads/20200111/20200111001827A2lTSP.png)

默认将生成的静态网站输出到 _book 目录。

也可以执行 gitbook build 命令构建书籍

```shell
gitbook build [书籍路径] [输出路径]
```

![](https://www.choudalao.com/uploads/20200111/20200111002227dhpPUR.png)

serve 命令也可以指定端口：

```shell
gitbook serve --port 250
```

生成 PDF 格式的电子书：

```shell
gitbook pdf ./../mybook.pdf
```

生成 epub 格式的电子书：

```shell
gitbook epub ./../mybook.epub
```

生成 mobi 格式的电子书：

```shell
gitbook mobi ./../mybook.mobi
```
