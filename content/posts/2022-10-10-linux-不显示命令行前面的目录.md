# 问题
当我们要到比较深的目录操作时，命令行光标前面总是有一串长长的目录，看着十分难受。

# 解决
对于有洁癖的人来说，看着极为难受，所以，我把它隐藏了，操作如下：

#### 创建".bash_profile"文件
```go
vim ~/.bash_profil
```

#### 文件内容输入
```go
export PS1='[\u@\h \W]\$ '
```

#### 重新加载内容
```go
source ~/.bash_profile 
```
