# 问题

本来想优化一下代码，写法如下：
```golang
if page, err := art.GetInt("page"); err != nil {
page = 1
}
limit, err := art.GetInt("limit")

offset := (page - 1) * limit
```

![](https://www.choudalao.com/uploads/20220416/20220416005502NxgigC.png)

编辑器提示`page`变量存在异常，在执行过程中也会报错，百思不得其解，查资料后才知道，原来啊，`if` 条件中的变量是临时的，值存在于当前 `if esle`语句内，外部是读取不到的。
