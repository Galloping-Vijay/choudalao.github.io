# 配置
# 放开对应安全组中Redis端口

在各大平台中把`redis`的端口加入安全组开放列表中。

# 修改 redis 配置

```php
# 设置密码
requirepass 12345
# 注释掉bind
#bind 127.0.0.1
# 默认不是守护进程方式运行，这里可以修改
daemonize no
# 禁用保护模式 把 yes 改成 no
protected-mode no
```
