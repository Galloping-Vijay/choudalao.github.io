# Scrapy的安装介绍

## windows
查看win下python版本
![](https://www.choudalao.com/uploads/20191124/20191124102716VnIiWn.png)

升级pip版本：`pip install --upgrade pip`

通过pip 安装 Scrapy 框架 `pip install Scrapy`

![](https://www.choudalao.com/uploads/20191124/201911241028156L6hzq.png)

## ubuntu

如果没有安装pip3先安装,`sudo apt install python3-pip`.
安装非Python的依赖 `sudo apt-get install python-dev python-pip libxml2-dev libxslt1-dev zlib1g-dev libffi-dev libssl-dev`

```shell
sudo pip3 install scrapy
```

![](https://www.choudalao.com/uploads/20191124/20191124122502swN153.png)

查看版本 `scrapy version`

![](https://www.choudalao.com/uploads/20191124/20191124122546OftN9l.png)
