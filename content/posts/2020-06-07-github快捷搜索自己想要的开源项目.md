###### 在项目名称name中搜索关键词
```shell
in:name laravel-wjfcms
```
![](https://www.choudalao.com/uploads/20200607/20200607005616J5Y7nm.png)

###### 在readme中搜索关键词
```shell
in:readme laravel-wjfcms
```
###### 获取start大于500
```shell
in:name laravel-wjfcms starts>500
```
###### 获取fork大于5000
```shell
fork>5000
```
###### 设置更新时间
```shell
pushed>2020-05-03
```
###### 设置编程语言
```shell
language:php
```
###### 在描述中搜索关键信息
```shell
in:description cms
```
###### 设置包大小（kb）
```shell
size:<=5000
```
###### 指定开源条款
```shell
license:apache-2.0
```
###### 综合练习
```shell
in:readme excel language:php starts>500
```
