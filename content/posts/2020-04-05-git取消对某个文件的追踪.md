# 取消追踪某个文件
# git取消文件跟踪
在使用git的时候，有些文件是不需要上传的，所以就可以修改   .gitignore 
例如：
如果是对所有文件都取消跟踪的话，就是
```go
git rm -r --cached . 　　//不删除本地文件
git rm -r --f . 　　//删除本地文件
```
 
# 对某个文件取消跟踪
```go
git rm --cached readme1.txt    删除readme1.txt的跟踪，并保留在本地。
git rm --f readme1.txt    删除readme1.txt的跟踪，并且删除本地文件。
```
