# 问题
laravel 在使用第三方登录时报如下错误
> Laravel \ Socialite \ Two \ InvalidStateException
No message

![](https://www.choudalao.com/uploads/20200722/20200722202739r8f9ws.png)

# 解决方法
在回调方法中添加 stateless()函数：stateless()函数的作用是禁止会话状态验证
```go
$socialUser = Socialite::with($account)->stateless()->user();
```
