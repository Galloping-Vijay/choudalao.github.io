## 快速开始

### 本地预览

```bash
# 启动预览服务器
./hugo.exe server

# 访问 http://localhost:1313
```

### 构建网站

```bash
# 构建静态网站
./hugo.exe

# 构建并最小化
./hugo.exe --minify
```
